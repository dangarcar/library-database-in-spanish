# Requisitos previos
- Tener Java

- Tener Maven

# How to build
Ejecuta el siguiente comando para hacer la build de la aplicación:

`mvn clean package`

Ejecuta el siguiente comando para ejecutar la aplicación:

`java -jar target/DatabaseInSpanish-(version).jar`
